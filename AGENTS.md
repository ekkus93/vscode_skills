# AGENTS.md

## Purpose
This repository expects agentic work to be careful, explicit, and repo-aware.

## Working rules
- Read relevant instructions before making changes.
- Use `.github/copilot-instructions.md` as the base policy.
- If the task maps to a shared workspace skill, consult `skills/SKILL_LIST.md` and the chosen skill's `SKILL.md`.
- Inspect adjacent files before editing.
- Prefer small, safe changes with clear rationale.

## Deliverables
For non-trivial tasks, provide:
- summary of findings
- concrete change plan
- risks
- tests to run
